# Assignment 13_12 — Tool / Action Guardrails

## Task

Create one prompt that uses tool / action guardrails and run it locally.

## Rules

Your prompt must include:
1. A clear role
2. A clear task
3. At least 4 tool / action guardrails
4. One action the assistant may recommend
5. One action the assistant must not claim to perform
6. A clear next-step response

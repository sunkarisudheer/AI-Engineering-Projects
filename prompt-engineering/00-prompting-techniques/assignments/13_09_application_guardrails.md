# Assignment 13_09 — Application-Level Guardrails

## Task

Create one workflow that applies an application-level guardrail before the model call.

## Rules

Your code must include:
1. One raw input text
2. One application-side guardrail step
3. One prompt built from the processed input
4. One final model response
5. Printed output showing the before and after effect

## Enterprise use cases to choose from

Choose any one:
- hospital message summarization assistant
- banking support note assistant
- emergency contact intake assistant
- telecom complaint summarization assistant
- insurance claim intake assistant

## Submission

Push your code to your GitHub repository and share the public repo link.

## What will be checked

- correct understanding of application-level guardrails
- clear separation between raw input and processed input
- simple and effective code
- clean output
- working execution

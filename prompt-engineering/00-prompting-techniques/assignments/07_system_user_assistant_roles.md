# Assignment 07 — System, User & Assistant Roles

## Task

Write one role-based prompt of your own and run it locally.

## Rules

1. Add a clear system instruction.
2. Add one user request.
3. Print both the system instruction and the user message before printing the model response.
4. Choose any one of these:
   - teacher
   - interviewer
   - customer support agent
   - writing coach
   - coding assistant

## Submission

Push your code to your GitHub repository and share the public repo link.

## What will be checked

- correct use of role-based prompting
- clarity of system instruction
- clarity of user request
- clean code
- working output

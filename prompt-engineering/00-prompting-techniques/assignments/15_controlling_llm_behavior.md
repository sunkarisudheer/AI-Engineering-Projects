# Assignment 15 — Controlling LLM Behavior

## Task

Create one prompt that controls the model’s behavior and run it locally.

## Rules

Your prompt must include:
1. A clear role
2. A clear task
3. At least 5 behavior-control rules
4. A realistic business or operational use case
5. A final response that clearly reflects the intended behavior

## Enterprise use cases to choose from

Choose any one:
- executive briefing assistant
- policy communication assistant
- hospital operations update assistant
- cyber incident status assistant
- game moderation report assistant

## Submission

Push your code to your GitHub repository and share the public repo link.

## What will be checked

- correct understanding of behavior control
- realistic control rules
- clear tone and style shaping
- clean code
- working output

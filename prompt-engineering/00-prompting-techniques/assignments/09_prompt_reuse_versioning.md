# Assignment 09 — Prompt Reuse & Versioning

## Task

Create one prompt in at least 2 versions and run both locally.

## Rules

1. Use the same task in both versions.
2. Version 1 should be simple.
3. Version 2 should be improved and more structured.
4. You may choose any one task:
   - study plan
   - email drafting
   - learning roadmap
   - content rewriting
   - business ideas
   - fitness plan

## Submission

Push your code to your GitHub repository and share the public repo link.

## What will be checked

- correct understanding of reuse
- correct understanding of versioning
- meaningful improvement from v1 to v2
- clean code
- working output

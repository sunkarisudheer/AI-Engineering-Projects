# Assignment 13_13 — Combined Guardrail Workflow

## Task

Create one assistant workflow that combines multiple guardrails in one prompt and run it locally.

## Rules

Your prompt must include:
1. A clear role
2. A clear task
3. At least 4 different guardrail types
4. A realistic enterprise or operational use case
5. A structured final response

## Enterprise or critical-domain use cases to choose from

Choose any one:
- workplace safety incident intake assistant
- hospital incident escalation assistant
- aviation operations alert assistant
- cybersecurity incident triage assistant
- disaster response coordination assistant

## Submission

Push your code to your GitHub repository and share the public repo link.

## What will be checked

- correct understanding of multiple guardrails together
- clear separation of guardrail types
- realistic workflow design
- clean code
- working output

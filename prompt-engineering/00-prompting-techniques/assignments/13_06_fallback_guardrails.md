# Assignment 13_06 — Fallback Guardrails

## Task

Create one prompt that uses fallback guardrails and run it locally.

## Rules

Your prompt must include:
1. A clear role
2. A clear task
3. At least 4 fallback rules
4. A case where the assistant should not answer normally
5. A safe next-step or escalation path

## Enterprise use cases to choose from

Choose any one:
- contract review intake assistant
- procurement exception review assistant
- policy clarification assistant
- internal audit intake assistant
- third-party risk triage assistant

## Submission

Push your code to your GitHub repository and share the public repo link.

## What will be checked

- correct understanding of fallback guardrails
- realistic fallback behavior
- useful escalation or clarification path
- clean code
- working output

# Assignment 02 — One-shot Prompting

## Task

Write one one-shot prompt of your own and run it locally.

## Rules

1. Give exactly one example.
2. After the example, ask the model to do the same task on a new input.
3. Choose any one of these:
   - tone conversion
   - title formatting
   - caption cleanup
   - simple rewriting
   - classification style alignment

## Submission

Push your code to your GitHub repository and share the public repo link.

## What will be checked

- correct use of one-shot prompting
- clarity of example
- clean code
- working output

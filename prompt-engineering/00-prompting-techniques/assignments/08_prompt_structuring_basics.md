# Assignment 08 — Prompt Structuring Basics

## Task

Write one structured prompt of your own and run it locally.

## Rules

Your prompt must include these 5 sections:
1. Role
2. Task
3. Context
4. Constraints
5. Output Format

## Choose any one topic

- study plan
- email drafting
- content rewrite
- simple business ideas
- beginner fitness routine
- learning roadmap

## Submission

Push your code to your GitHub repository and share the public repo link.

## What will be checked

- correct prompt structure
- clarity of each section
- useful output format
- clean code
- working output

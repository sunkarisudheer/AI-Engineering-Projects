# Assignment 01 — Zero-shot Prompting

## Task

Write one zero-shot prompt of your own and run it locally.

## Rules

1. Do not give the model any example.
2. Ask only one clear task.
3. Choose any one of these:
   - summarization
   - rewriting
   - translation
   - idea generation
   - simple classification

## Submission

Push your code to your GitHub repository and share the public repo link.

## What will be checked

- correct use of zero-shot prompting
- clarity of instruction
- clean code
- working output

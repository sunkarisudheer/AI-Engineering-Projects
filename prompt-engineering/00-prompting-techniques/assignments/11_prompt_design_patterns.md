# Assignment 11 — Prompt Design Patterns

## Task

Choose one piece of input text and apply at least 3 different prompt design patterns to it.

## Rules

1. Use the same input text for all patterns
2. Apply at least 3 of these:
   - classification
   - extraction
   - transformation
   - generation
3. Print each output clearly

## Suggested input types

- product review
- email message
- short article
- feedback note
- support complaint

## Submission

Push your code to your GitHub repository and share the public repo link.

## What will be checked

- correct understanding of design patterns
- clear difference between pattern outputs
- clean code
- working output

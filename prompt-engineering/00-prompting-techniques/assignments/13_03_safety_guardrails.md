# Assignment 13_03 — Safety Guardrails

## Task

Create one prompt that uses safety guardrails and run it locally.

## Rules

Your prompt must include:
1. A clear role
2. A list of allowed safe requests
3. A list of blocked unsafe requests
4. A refusal behavior for unsafe requests
5. A safe alternative or redirect message
6. Structured output

## Enterprise use cases to choose from

Choose any one:
- enterprise security support assistant
- internal compliance FAQ assistant
- workplace safety incident assistant
- financial fraud awareness assistant
- regulated healthcare triage information assistant

## Submission

Push your code to your GitHub repository and share the public repo link.

## What will be checked

- clear understanding of safety guardrails
- realistic blocked categories
- useful refusal behavior
- safe redirect or alternative
- clean code
- working output

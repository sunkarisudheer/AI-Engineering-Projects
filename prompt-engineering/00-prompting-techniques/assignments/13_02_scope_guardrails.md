# Assignment 13_02 — Scope Guardrails

## Task

Create one prompt that uses scope guardrails and run it locally.

## Rules

Your prompt must include:
1. A clear role
2. A clear allowed scope
3. A clear out-of-scope list
4. A fallback response for out-of-scope questions
5. At least one in-scope question and one out-of-scope question

## Suggested use cases

Choose any one:
- travel expense assistant
- leave policy assistant
- interview scheduling assistant
- product return assistant
- employee onboarding FAQ assistant

## Submission

Push your code to your GitHub repository and share the public repo link.

## What will be checked

- correct understanding of scope guardrails
- clear allowed and out-of-scope boundaries
- useful fallback message
- clean code
- working output

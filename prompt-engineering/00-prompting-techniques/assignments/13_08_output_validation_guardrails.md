# Assignment 13_08 — Output Validation Guardrails

## Task

Create one workflow that validates the model output after generation.

## Rules

Your code must include:
1. A prompt with a required structure
2. A model response
3. A validation step after the response
4. At least 3 required output sections
5. A clear pass/fail validation message

## Enterprise or operational use cases to choose from

Choose any one:
- emergency response incident summary assistant
- hospital triage summary assistant
- military field status report assistant
- aviation incident note assistant
- cybersecurity alert summary assistant

## Submission

Push your code to your GitHub repository and share the public repo link.

## What will be checked

- correct understanding of output validation guardrails
- clear validation logic
- correct post-response checking
- clean code
- working output

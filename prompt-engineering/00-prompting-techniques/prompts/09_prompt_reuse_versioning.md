# 09 — Prompt Reuse & Versioning

## What it means

Prompt reuse means using the same prompt structure again for similar tasks.

Prompt versioning means improving a prompt over time and saving the improved versions clearly.

## Why this matters

In real projects, prompts are not written only once.
They are often:
- reused
- refined
- compared
- improved

## Simple example

- Prompt v1 = basic
- Prompt v2 = clearer and more structured
- Prompt v3 = even more controlled

## Reuse example

A learning-plan prompt can be reused by changing:
- topic
- audience
- time available

## Key idea

Good prompts are often built in versions, not in one attempt.

# 08 — Prompt Structuring Basics

## What it means

Prompt structuring means organizing a prompt into clear sections instead of writing everything in one unclear paragraph.

## A simple beginner structure

### 1. Role
Defines who the model should act like.

### 2. Task
Defines what the model should do.

### 3. Context
Provides background information.

### 4. Constraints
Sets rules, limits, or boundaries.

### 5. Output Format
Defines how the final answer should look.

## Why this matters

Structured prompts are:
- easier to understand
- easier to control
- easier to reuse
- more reliable than messy prompts

## Key idea

A well-structured prompt usually gives better output than a vague prompt.

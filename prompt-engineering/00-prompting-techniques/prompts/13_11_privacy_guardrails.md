# Assignment 13_11 — Privacy Guardrails

## Task

Create one prompt that uses privacy guardrails and run it locally.

## Rules

Your prompt must include:
1. A clear role
2. A clear task
3. At least 4 privacy guardrails
4. Input text that contains sensitive or personal data
5. A final response that avoids unnecessary disclosure

## Enterprise use cases to choose from

Choose any one:
- banking support summary assistant
- insurance claims summary assistant
- hospital operations note assistant
- HR employee case summary assistant
- telecom customer complaint assistant

## Submission

Push your code to your GitHub repository and share the public repo link.

- correct understanding of privacy guardrails

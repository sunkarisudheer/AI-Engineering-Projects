# Prompting Techniques Foundation

This project contains simple beginner-friendly examples for learning core prompting techniques with Python and the Gemini API.

## Topics Covered

- Zero-shot prompting
- One-shot prompting
- Few-shot prompting
- Multi-shot prompting
- Chain-of-Thought prompting
- Zero-shot Chain-of-Thought prompting

## Project Structure

- `examples/` → runnable Python examples
- `prompts/` → concept notes for each prompting technique
- `assignments/` → practice tasks for learners
- `helper.py` → reusable helper function for API calls
- `.env.example` → sample environment variable format

## Setup

1. Create a virtual environment
2. Install dependencies from `requirements.txt`
3. Create a `.env` file
4. Add your Gemini API key in `.env`

Example:

```env
GEMINI_API_KEY=your_api_key_here

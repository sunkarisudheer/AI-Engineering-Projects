# Student Assignment Overview

This project contains beginner-friendly practice exercises for common prompting techniques.

## Included techniques
- Zero-shot prompting
- One-shot prompting
- Few-shot prompting
- Multi-shot prompting
- Chain-of-Thought prompting
- Zero-shot Chain-of-Thought prompting

## Submission expectation
Complete the assignment files inside the `assignments` folder, create your own local project, push your work to GitHub, and share the public repository link for review.

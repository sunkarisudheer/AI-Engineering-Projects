# Gemini Chatbot using Angular and Python

A simple full-stack AI chatbot application built using **Angular** for the frontend and **Python Flask** for the backend, powered by the **Google Gemini API**.

This project allows users to enter messages in a chat interface and receive AI-generated responses in real time.

---

## Features

- Modern chat UI built with Angular
- Python Flask backend API
- Integration with Google Gemini API
- Secure API key handling using `.env`
- Real-time message sending and response display
- Loading state support for better user experience
- Clean component-based frontend structure

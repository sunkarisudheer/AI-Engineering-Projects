## 📂 Repository Structure

### 1️⃣ Personal Expense Tracker

A command-line based Python application that helps users track and manage their daily expenses.

**Features**
* Add expenses with category and amount
* View all recorded expenses
* Calculate total spending
* Display category-wise expense summary
* Save and load expense data from storage

**Tech Stack**
* Python
* File Handling
* Modular Programming

---

### 2️⃣ Tokenizer

A project that demonstrates how **tokenization works in Natural Language Processing (NLP)** using different pretrained tokenizers.

**Features**
* Input text and generate tokens
* Compare tokenization across multiple models
* Display token IDs and token representations
* Visualize tokens with colored output for clarity

**Concepts Covered**
* NLP Tokenization
* Subword Tokenization
* Token IDs
* Pretrained Tokenizers

**Libraries Used**
* Python
* HuggingFace Transformers
